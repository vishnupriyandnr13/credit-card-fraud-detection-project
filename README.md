# Credit Card Fraud Detection Using Machine Learning

## Overview
This project focuses on detecting fraudulent credit card transactions using machine learning techniques. The model is trained on anonymized transaction data to classify transactions as legitimate or fraudulent.

This project was completed as part of an IBM Cognitive Class guided project and enhanced with improved project organization and documentation.


## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Fraud transaction classification
- Machine learning model training
- Model evaluation and accuracy comparison
- Visualization of results

---

The following models were used in this project:

## Dataset
The project uses a credit card transaction dataset containing anonymized transaction details.

Due to GitHub file size limitations, the dataset is not included in this repository.

Download the dataset from:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

After downloading, place:

```text
creditcard.csv
```

inside the `dataset/` folder.

## Installation

Install required dependencies:

```bash
pip install -r requirements.txt
```


## How to Run the Project

1. Clone the repository
2. Download the dataset
3. Place `creditcard.csv` inside the `dataset/` folder
4. Open the notebook in Jupyter Notebook or Google Colab
5. Run all notebook cells
